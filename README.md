# 📚 불리언 연산자 실습 플랫폼 (Boolean Simulator)

> 기존 단순 클라이언트 측 검색 체계를 넘어선, **Python 기반의 Streamlit 웹 애플리케이션**입니다. 학생 정보 스토리지(DB) 관리, 깐깐한 대소문자 검증, 최종 오답노트 생성 체계를 갖춘 **고등학교 탐구/수행평가 특화 모바일 대응 실습 플랫폼**입니다.

---

## 📁 디렉토리 및 파일 구성

```text
bool/
├── app.py          ← [메인 스트림릿 앱] 모바일 맞춤 라우팅 및 뷰 구성 컨트롤러
├── logic.py        ← [판독 엔진] AST(추상 구문 트리) 파서 및 맞춤형 오개념 진단 로직
├── backend.py      ← [DB 백엔드] SQLite3 및 Pandas 기반 성적 누적 시스템
├── data.py         ← [데이터] 24종의 가상 학술 논문 및 실습 퀴즈 데이터셋
├── students.db     ← [자동 생성] 접속/채점 기록이 저장되는 로컬 데이터베이스
└── README.md       ← 이 파일
```

---

## 🚀 실행 및 배포 방법

이 앱은 파이썬 가상환경 위에서 구동됩니다. 터미널(또는 명령 프롬프트)에서 아래 명령을 실행하세요.

### 1단계: 패키지 설치 (초기 1회)
```bash
pip install streamlit pandas
```

### 2단계: 로컬 서버 구동
```bash
streamlit run app.py
```
> 서버가 실행된 후 터미널에 뜨는 `Network URL (예: http://192.168.0.x:8501)` 주소를 학생들에게 배포하시면, 각자의 스마트폰을 통해 앱에 접속할 수 있습니다.

## 🖥️ 주요 기능 및 특장점

### 1. 🎓 모바일 최적화 실습 환경 (연구 과제 수행 탭)
- **모바일 폼팩터 UI**: 거추장스러운 사이드바를 버리고 상단 탭 내비게이션 바(`radio`)와 중앙 정렬 레이아웃(`layout="centered"`)을 도입하여 **스마트폰** 화면에서 완벽한 웹앱 비율을 구현했습니다.
- **상황 기반의 문제 해결**: "생활과 윤리 보고서에 칸트와 의무론이 동시에 들어간 논문 찾기" 등 실제 겪을 법한 시나리오를 8문항 제공합니다.
- **학생 빈출 오류(오개념) 방지**: 단순히 정답 결과를 맞히는 것이 아니라, 괄호 위치, 따옴표 유무, 소문자 입력, 묵시적 연산 생략 여부를 **Token 단계에서 깐깐하게 검증**합니다. (반드시 대문자 `AND`, `OR`, `NOT`만 허용)

### 2. 📝 지연된 평가 (Deferred Evaluation) 체계
- **즉각 힌트 제거**: 제출 버튼을 눌러도 바로 정답 여부를 알려주어 찍어 맞히기(Trial & Error)를 하는 꼼수를 원천 차단했습니다.
- **맞춤형 오답 노트 (성적 대시보드)**: 모든 문항(8개)이 끝난 직후 표(Pandas DataFrame)가 나타납니다. 학생은 자기가 입력한 쿼리가 무엇인지 되돌아보고, 어떤 부분을 실수했는지 교수님의 **[상세 진단 피드백]**을 한눈에 확인할 수 있습니다.

### 3. 👨‍🏫 관리자 기능 (Admin 탭)
- 초기 로그인 화면에서 학번 란에 **`admin`**을 입력하고 들어오면 숨겨진 관리자 탭이 활성화됩니다.
- 비밀번호(`admin1234`)를 입력하면 현재 접속했던 학생들의 학번, 이름, 평균 퀴즈 점수, 접속 현황을 **액셀 표 형식**으로 언제든 다운로드 및 조회가 가능합니다.

### 4. 🔎 검색 연산자 가이드 (가이드 탭)
- 논문 검색 시 가장 많이 쓰이는 5가지 도구(`AND`, `OR`, `NOT`, `( )`, `" "`)에 대한 작동 원리와 예시를 명확하고 짧게 제시합니다. 언제든 상단 탭을 눌러 컨닝(?)하며 공부할 수 있습니다.

## 🎨 사용 기술 스택

- **언어 및 런타임**: Python 3.10+
- **웹 프레임워크**: Streamlit (Reactive UI, Session State 캡슐화)
- **데이터 처리**: SQLite3 (경량 RDBMS), Pandas (결과 렌더링 및 통계)
- **추론 및 파싱 구역**: 정규식 파서, Shunting-yard (중위표기→후위표기) 기반 AST 평가 트리
